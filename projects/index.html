OSS & Hobby Projects

3D Color Space

3D Color Space - An HTML/JS tool for visualizing color spaces in 3D. It allows to see how each color is distributed in three-dimensional space. This can be just fun to use. It can be used to visually explain color distribution and why an how we have different color clusters. Finally, it can provide very visual feedback on what are the prevalent colors in the image/icon and be helpful. It can essentially create a color palette based on an image.

Rust Book

Implemented fully compliant EPUB format generation pipeline (in Rust, of course) with ability to reflow and optimize content for reading on smaller screens (primarily mobile devices) for the fundamental The Rust Programming Language ebook. It is a great resource and I wanted to make it more accessible for people who want to learn Rust on the go. Check out the full story and the walking talking Rusty the Rustacean.

Domainator

Domainator is a fast, versatile, and unified CLI domain availability checker. Built to run perfectly under Bun (or Node.js), it accepts standard input, domain expansion, and piped lists, delivering rich availability checks via RDAP, DNS DoH, WHOIS fallbacks, and multi-registrar API checks.
