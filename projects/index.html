OSS and Hobby Projects

3D Color Space

3D Color Space - A small tool for visualizing color spaces in 3D. It allows you to see how different colors relate to each other in a three-dimensional space, which can be helpful for understanding color theory and for choosing color schemes.

Rust Book

Rust Programming Language Book Implemented fully compliant EPUB format generation (in Rust, of course) with ability to reflow and optimize content for reading on smaller screens (mobile devices). It is a great book and I wanted to make it more accessible for people who want to learn Rust on the go.
