Preview any File on OS X in Finder with QuickLook Plugin

17 Apr 2015

Here's an excellent plugin to preview any arbitrary file as a text file.

Very handy for previewing README or any other weird extension files that finder would not preview by default.

Once you unpack the archive, you need to move the plugin to ~/Library/QuickLook. To get there fast (or any folder for that matter) you can use a nifty Finder shortcut: ⌘ + ⇧ + G
