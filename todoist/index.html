Todoist Tips, Shortcuts etc...

Shortcuts

  • Attach a label — type @ followed by the label name
  • Set a priority level — type p followed by the priority number (1–4)
  • Organize the task into a project — type # followed by the project name
  • Assign the task to a coworker — type + followed by the collaborator’s name
